# Tunexori
Tunexori is a lightweight Windows internet radio player for direct stream URLs. It lets you add your own radio
  streams, control playback from the main window or the system tray, view track metadata when available, and use extra
  tools such as an alarm, sleep timer, and track history.
  ## Screenshot
  ![Tunexori interface1](assets/tunexori-interface1.png)
  ## Features

  - Add and manage custom internet radio stream URLs.
  - Play, pause, and switch between stations.
  - Control playback from the main app window or the Windows system tray.
  - Keep playback running while the window is minimized to the tray.
  - Adjust volume directly from the interface.
  - View a live audio level visualizer.
  - Use an alarm to start the current station at a selected time.
  - Use a sleep timer to pause playback after a selected number of minutes.
  - Save track history for streams that provide ICY metadata.
  - Export track history to a text file.
  - Copy the current track title.
  - Enable startup with Windows.
  - Start the last selected station automatically when the app opens.
  - Check for updates through GitHub Releases.

  ## Requirements

  To run Tunexori.exe directly, you need:

  - Windows 10 or Windows 11 x64
  - .NET Desktop Runtime 9 x64
  - Microsoft Edge WebView2 Runtime x64

  Download links:

  - .NET Desktop Runtime 9 x64:
    https://aka.ms/dotnet/9.0/windowsdesktop-runtime-win-x64.exe
  - Microsoft Edge WebView2 Runtime x64

    WebView2 Runtime is usually already included or automatically available on modern Windows 10 and Windows 11 systems.
  This link is provided only as a fallback in case it is missing on your device:

    https://go.microsoft.com/fwlink/?linkid=2124701
